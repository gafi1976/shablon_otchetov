# Генератор актов — Списание и Установка оборудования

Графическое приложение на Python (Tkinter) для автоматической генерации
красивых Word-документов актов **списания** и **ввода в эксплуатацию** оборудования.

---

## Структура файлов

```
shablon_otchetov/
├── app.py                  — главное приложение (Tkinter GUI)
├── generator_spisan.py     — генератор Word актов списания
├── generator_ust.py        — генератор Word актов установки
├── excel_handler.py        — чтение/создание Excel шаблонов
├── requirements.txt        — зависимости Python
└── README.md
```

---

## Установка

```bash
pip install -r requirements.txt
```

> **Требования:** Python 3.9+, tkinter (входит в стандартную поставку Python)

---

## Запуск

```bash
python app.py
```

## Возможности

### Вкладка «Списание»
- Заполнение данных организации и комиссии
- Ввод списка оборудования через таблицу (двойной клик — редактирование)
- Генерация красивого Word акта списания с:
  - Грифом «УТВЕРЖДАЮ»
  - Составом комиссии
  - Таблицей оборудования (тёмно-синяя шапка, чередующиеся строки)
  - Заключением комиссии
  - Блоком подписей
- **Загрузка данных из Excel шаблона**
- **Создание Excel шаблона** для заполнения

### Вкладка «Установка»
- Всё то же самое для акта ввода в эксплуатацию
- Дополнительное поле «Место установки»
- Цветовая метка состояния оборудования (Новый — зелёный, Б/У — оранжевый)

### Вкладка «Пакетная генерация»
- Добавление нескольких Excel файлов за раз
- Генерация всех актов одним кликом

### Режимы сохранения
| Режим | Описание |
|-------|----------|
| **Каждый акт — отдельный файл** | `Akt_Spisaniya_001.docx`, `Akt_Spisaniya_002.docx` ... |
| **Все акты в одном файле** | `Akty_Spisaniya_vse_20240615.docx` |

## Формат Excel шаблонов

### Шаблон списания (`shablon_spisaniya.xlsx`)
Создаётся кнопкой «Создать шаблон Excel» во вкладке «Списание».

Поля организации (строки 3–9):
- Организация, Регион, Номер акта, Дата, Председатель, Члены комиссии

Данные оборудования (с строки 12):
| № | Наименование | Инв. номер | Год | Перв. ст-сть | Ост. ст-сть | Причина | Примечание |

### Шаблон установки (`shablon_ustanovki.xlsx`)
Поля организации (строки 3–10): + поле «Место установки»

Данные оборудования (с строки 13):
| № | Наименование | Модель | Серийный № | Инв. номер | Год | Стоимость | Дата уст-ки | Состояние | Примечание |
